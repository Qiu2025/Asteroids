# Asteroids
Microgame developed to consolidate the basic aspects of video game development with Unity in the **FdV** course at **UPM**.

The player controls a spaceship that can rotate, thrust and shoot to destroy incoming asteroids.

## Demo
You can play a hosted build of the game here:
[Unity Play](https://play.unity.com/es/games/b589b4fe-6c14-42b2-8441-2a438c68c7c7/asteroids)

## Installation
### Prerequisites
- Unity Editor
- Git

### Steps
1. Clone the repository  
   `git clone https://github.com/Qiu2025/Asteroids.git`
2. Open Unity Hub and add the cloned folder as a new project
3. Open it (dependencies will be imported automatically)
4. Press the Play button in the Unity Editor to run the game
